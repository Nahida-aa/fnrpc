# `@fnrpc/query-core` — DEPRECATED

This package was an intermediate attempt at a framework-agnostic TanStack Query integration layer for fnRPC. After refactoring, it is **no longer used**.

Functionality has been superseded by `@fnrpc/tanstack-query` (framework-agnostic) and framework-specific packages (`@fnrpc/solid-query`, etc.).

**No further development.** Kept for historical reference only.
